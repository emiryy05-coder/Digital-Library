--Dijital Kütüphane Yönetim Sistemi
Bu proje, kitap envanterini yönetmek, kategorilere göre filtrelemek ve ödünç işlemlerini takip etmek için geliştirilmiş, kullanıcı dostu bir Web Uygulamasıdır. İlk web geliştirme projem olarak, modern arayüz tasarımı ve dinamik veri yönetimi konularına odaklanmıştır.

--Özellikler
Dinamik Kategori Filtreleme: Roman, Polisiye ve Tarih gibi kategoriler arasında kolay geçiş.

Envanter Takibi: Toplam kitap sayısı ve ödünç verilen kitapların anlık takibi.

Ödünç Alma Sistemi: Kitapların ödünç verilme durumuna göre tablolarda otomatik güncellenme.

Admin Paneli: Kitap ekleme, silme ve kullanıcı işlemlerinin yönetildiği özel alan.

Local Storage Entegrasyonu: Verilerin tarayıcı hafızasında saklanması (sayfa yenilense de veriler kaybolmaz).

Responsive Tasarım: Mobil ve masaüstü cihazlarla tam uyumlu arayüz.

--Kullanılan Teknolojiler
HTML5: Sayfa yapısı ve içerik yönetimi.

CSS3 & Bootstrap 5: Modern ve şık arayüz tasarımı, gölgelendirme efektleri (Gemini desteğiyle optimize edildi).

JavaScript (ES6+): Dinamik içerik yönetimi, Local Storage işlemleri ve form kontrolleri.
